## 环境及开发
01. cd git path
02. npm install hexo --save
03. npm i --save hexo-wordcount
04. hexo -v
05. hexo init
06. npm install
07. hexo g
08. npm install hexo-deployer-git --save
09. hexo new
10. hexo clean
11. hexo d -g
12. hexo s
13. hexo deploy
14. Front-matter
